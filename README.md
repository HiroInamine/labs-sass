# Laboratorio SASS (Syntactically Awesome StyleSheets)

## Instalacao
Install Ruby
Install SASS
http://sass-lang.com/install

## Como usar

```
sass .\sass\main.scss .\css\main.css
```

OU

```
sass --watch  .\sass\main.scss:.\css\main.css
```
